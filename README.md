# network-scanning-nmap
Network Scanning and observation using Nmap

Project: Network Scanning using Nmap

Objective :
Identify open ports and services on a vulnerable machine.

Target :
Metasploitable 2

Tool :
Nmap

Commands Used :
nmap target-ip
nmap -sV target-ip
nmap -O target-ip
nmap -A target-ip

Results :
Multiple open ports discovered including FTP, SSH, Telnet, HTTP, and MySQL.

Conclusion :
The scan revealed several exposed services which could be further analyzed for vulnerabilities.
